# Blaster-Master-GBC-Audio-Disasm
Blaster Master: Enemy Below (GBC) Audio Disasm

A disassembly of the sound engine and data from Blaster Master: Enemy Below for Game Boy Color, fully labeled and commented. Can be built with RGBDS; the sound data of the resulting ROM is verified to match the sound data from the original ROM exactly. The resulting ROM includes a custom player where all the music and sound effects can be heard.
A text file is also included which documents how the music, sound effect, and instrument data is structured for this game as well as other games using versions of the same sound engine. This game uses a later version of the sound engine in Mega Man II, which a full disassembly (not by me) can be found at https://github.com/Kak2X/rw2.
